


Tumbo is a small and simple templated matrix library.

Features
* templated matrix type that is given a size at compile time
* no dynamic allocations


Install
Either copy the header files to where ever you want or use cmake.

CMake
make a build/ directory and change to it. Type "cmake ../ && make install" and
you're done. If you wish to install at a specific location, pass
"-DCMAKE_INSTALL_PREFIX=/MyLocation/" as an argument to CMake.
